# 贪吃蛇大冒险 (Snake Adventure)

网页版闯关贪吃蛇，依据 `prd.md` 实现：经典吃豆循环 + 10 关渐进机制。

## 运行方式

本项目为纯静态前端（HTML / CSS / ES Module），**无需构建**。

```bash
# 任意静态服务器即可，例如：
npx --yes serve .
# 或
python3 -m http.server 8080
```

浏览器打开提示的本地地址即可。因使用 ES Module，请勿直接用 `file://` 打开。

## 操作

| 按键 | 功能 |
| :--- | :--- |
| 方向键 | 转向（带输入队列，禁止瞬间 180° 掉头） |
| 空格 | 暂停 / 继续 |
| R | 重开当前关卡 |

## 关卡概览

1. 初探 — 基础教学  
2. 碎石迷宫 — 静态石块  
3. 狭路相逢 — 长条墙  
4. 穿墙奇术 — 边界穿墙  
5. 减速泥潭 — 泥潭减速 50%  
6. 移动挡板 — 水平往返障碍  
7. 生命危机 — 尖刺 + 3 命  
8. 镜像世界 — 周期性反向控制  
9. 时间竞速 — 60 秒倒计时  
10. 最终试炼 — 综合地形 + AI 红蛇
